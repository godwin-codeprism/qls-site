Welcome to QucikLearn Systems Admin Panel

Here is all you need to know

  • You can find all your editabled stuff of your website under   Componenets tab in menu on the left

  • Click to shrink side menu and gain more working area

  • Click to add a new row just below the current row

  • Click to delete the current row

    • Click if you are sure of deleting the data in current row permanently
    • Click if you are not sure of deleting the data in current row

  • Click to edit or change contents in current row.

    • All the text cells will turn into editable fileds

    • Click to update links

    • Click to update the current image.

    • Click to remove current image.
      (A palceholder image will appear if current image is removed)

  • Make sure to click save once you are done with editing data of current row

  • Click to go to QuickLearn Systems Homepage

  • Click to Logout
